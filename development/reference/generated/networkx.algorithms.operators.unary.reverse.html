reverse¶

reverse(G, copy=True)[source]¶

Return the reverse directed graph of G.

Parameters :

G : directed graph

A NetworkX directed graph

copy : bool

If True, then a new graph is returned. If False, then the graph is reversed in place.
Returns :

H : directed graph

The reversed G.
