__contains__¶

MultiGraph.__contains__(n)¶

Return True if n is a node, False otherwise. Use the expression ‘n in G’.

Examples

>>> G = nx.Graph()   # or DiGraph, MultiGraph, MultiDiGraph, etc
>>> G.add_path([0,1,2,3])
>>> 1 in G
True
