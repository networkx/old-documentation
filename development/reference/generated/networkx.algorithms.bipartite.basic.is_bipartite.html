is_bipartite¶

is_bipartite(G)[source]¶

Returns True if graph G is bipartite, False if not.

Parameters :G : NetworkX graph

See also

color, is_bipartite_node_set

Examples

>>> from networkx.algorithms import bipartite
>>> G = nx.path_graph(4)
>>> print(bipartite.is_bipartite(G))
True
