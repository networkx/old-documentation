average_node_connectivity¶

average_node_connectivity(G)[source]¶

Returns the average connectivity of a graph G.

The average connectivity \(\bar{\kappa}\) of a graph G is the average of local node connectivity over all pairs of nodes of G [R205] .

\[\bar{\kappa}(G) = \frac{\sum_{u,v} \kappa_{G}(u,v)}{{n \choose 2}}\]
Parameters :

G : NetworkX graph

Undirected graph
Returns :

K : float

Average node connectivity
