node_link_graph¶

node_link_graph(data, directed=False, multigraph=True, attrs={'source': 'source', 'target': 'target', 'key': 'key', 'id': 'id'})[source]¶

Return graph from node-link data format.

Parameters:

data : dict

node-link formatted graph data

directed : bool

If True, and direction not specified in data, return a directed graph.

multigraph : bool

If True, and multigraph not specified in data, return a multigraph.

attrs : dict

A dictionary that contains four keys ‘id’, ‘source’, ‘target’ and ‘key’. The corresponding values provide the attribute names for storing NetworkX-internal graph data. Default value: dict(id='id', source='source', target='target', key='key').
Returns:

G : NetworkX graph

A NetworkX graph object

See also

node_link_data, adjacency_data, tree_data

Notes

The default value of attrs will be changed in a future release of NetworkX.

Examples

>>> from networkx.readwrite import json_graph
>>> G = nx.Graph([(1,2)])
>>> data = json_graph.node_link_data(G)
>>> H = json_graph.node_link_graph(data)
